* **Key Library:** NumPy / TensorFlow.

### 2. [CNN](./CNN) - Convolutional Neural Networks
Designed for spatial data (images). I explored how filters "slide" over images to detect patterns like edges and shapes.


[Image of convolutional neural network architecture]

* **Project:** Image recognition using CIFAR-10.
* **Key Concept:** Max Pooling & Dropout for preventing overfitting.

### 3. [RNN](./RNN) - Recurrent Neural Networks
Specialized for data where order matters (text or time-series). 

* **Focus:** Understanding why standard RNNs struggle with "long-term memory" and how LSTMs solve it.
